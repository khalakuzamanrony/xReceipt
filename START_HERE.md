## ⚡ Quick Start (2 minutes)

### Step 1: Open Templates Page
Navigate to **Templates** in the sidebar

### Step 2: Click "Template Builder"
You'll see 4 pre-designed templates

### Step 3: Select a Template
Choose one that matches your needs:
- **Minimal** - Simple & clean
- **Professional** - Business invoice
- **Modern** - Colorful & contemporary
- **Compact** - For thermal printers

### Step 4: Create Template
1. Click "Use Template"
2. Enter a custom name
3. Click "Create Template"
4. Done! Your template is ready to use

## 🚀 How to Use

### Creating a Template
```
Templates → Template Builder → Select Template → 
Enter Name → Create Template → Done!
```

### Using Template in Receipt
```
Receipts → Create Receipt → Select Your Template → 
Fill Data → Preview → Send/Print
```

### Customizing a Template
```
Templates → Find Template → Edit → Modify HTML → 
Update → Done!
```

## 💡 Template Variables

Use these in your templates:
```
{{RECEIPT_ID}}      - Receipt number
{{DATE}}            - Transaction date
{{CUSTOMER_NAME}}   - Customer name
{{CUSTOMER_EMAIL}}  - Customer email
{{ITEMS}}           - Item list
{{TOTAL}}           - Total amount
{{SUBTOTAL}}        - Subtotal
{{TAX}}             - Tax amount
{{STATUS}}          - Payment status
{{COMPANY_NAME}}    - Your company
{{COMPANY_EMAIL}}   - Company email
{{FOOTER_MESSAGE}}  - Custom footer
```
